# PetNutrition - Сервис оптимизации рациона для домашних животных

**PetNutrition** - это веб-сервис для расчета и оптимизации рациона домашних животных. Система помогает владельцам питомцев подбирать идеальное питание на основе индивидуальных параметров: вида животного, возраста, веса, уровня активности и целей (похудение, поддержание веса, набор массы).
## Ссылка: https://petnutrition.onrender.com


## Основные возможности

### Расчет питания
### Возможность добавлять и удалять профили питомца
### Подбор корма
