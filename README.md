# PetNutrition - Сервис оптимизации рациона для домашних животных

Сервис для расчета суточной нормы питания домашних животных с учетом индивидуальных параметров и целей.

## Технологии
- Python 3.10+
- Django 5.2
- SQLite (разработка)
- Bootstrap 5
